These are static mockups with placeholder numbers — a visual direction, not built screens. The concept to react to: gold = the Live strategy, cyan = candidates, dimmed = benchmarks and controls — and no number stands alone. Five v5 honesty ideas are shown on purpose: an α (β-adj) column (Jensen's alpha with IR — so a low-beta strategy isn't rigged to lose and a high-beta one isn't flattered), dual Buy&Hold benchmarks (cap-weight for the alpha regression, equal-weight because the strategies are equal-weight portfolios), turnover-matched + cost-free Random rows (a fair noise floor and the pure-noise line), a health pill beside every return (Suspect vetoes promotion no matter how good the number), and an MDE line under the table (gaps smaller than the window can detect read "too early to judge").
Paper-trading / research tool. No real orders, no real funds. Not investment advice.
