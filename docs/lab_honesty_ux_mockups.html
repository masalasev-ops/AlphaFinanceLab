UI Mockups · v6 · UX honesty pass

The verdict leads. The evidence accrues on screen.

Companion to the v5 mockups — same chrome, palette, and plain names. This pass fixes one contradiction and adds the v6 machinery: the old screens looked like a leaderboard, but the lab's own statistics say most rank gaps are unjudgeable for months. So here, verdict chips outrank return numbers, unjudgeable alphas look unjudgeable, ranks become tiers, the 200-member random population appears as a band and a percentile, and the wait for significance is drawn as progress, not absence.

"Too early" used to be a dead end. The meter turns it into a journey: the target length comes straight from the MDE formula run backwards — how many days until the current gap clears the smallest-detectable bar. If the gap grows, the target date moves closer. Waiting is the product; show it accruing.
